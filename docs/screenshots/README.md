# Screenshot Guide / Guia de Screenshots

This folder is used by `README.md` and `README.pt-BR.md` for UI gallery images.

## Expected files

- `grafana-k8s-overview.svg`
- `grafana-gameplay-overview.svg`
- `grafana-logs-overview.svg`
- `argocd-applications.svg`
- `argocd-application-details.svg`

These `.svg` files are placeholders. Replace each one with a real screenshot (same filename), or update image links in the READMEs to `.png` files.

## Capture checklist (EN)

1. Open Grafana (`http://localhost:30030`) and Argo CD (`http://localhost:30080`).
2. Set browser zoom to 100% and use full-width layout.
3. Capture at 1600x900 or 1920x1080.
4. Recommended screenshots:
- Grafana dashboard `Terraria K8s Overview`
- Grafana dashboard `Terraria Gameplay Overview`
- Grafana dashboard `Terraria Logs Overview`
- Argo CD Applications list
- Argo CD Application details view
5. Save images using the expected names in this folder.
6. Commit and push.

## Checklist de captura (PT-BR)

1. Abra Grafana (`http://localhost:30030`) e Argo CD (`http://localhost:30080`).
2. Deixe zoom do navegador em 100% e layout em tela cheia.
3. Capture em 1600x900 ou 1920x1080.
4. Capturas recomendadas:
- Dashboard `Terraria K8s Overview`
- Dashboard `Terraria Gameplay Overview`
- Dashboard `Terraria Logs Overview`
- Lista de Applications do Argo CD
- Tela de detalhes de Application no Argo CD
5. Salve com os nomes esperados nesta pasta.
6. Commit e push.
